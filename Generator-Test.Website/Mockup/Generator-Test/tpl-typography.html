Typography

Type is all set with the rems, so font-sizes and spacial relationships can be responsively sized based on a single :root font-size property. Out of the box, we never changes the :root font-size, but it's there in case you need it. All measurements are still base 10 though so, 5.0rem font-size just means 50px.
